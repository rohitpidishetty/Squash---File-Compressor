# Thankyou for downloading the software.

Open terminal and use the following command to compress a file,
# ./squash -compress origin_file target_file_name
example: ./squash -compress "E:\\Folder\\Folder" ./arc.tar.sq

To decompress,
# ./squash -decompress target_file_name.tar.sq
example: ./squash -decompress ./arc.tar.sq

# Note: Only these file can be compressed

Plain text files
ASCII / Latin-1
Source code
Logs
README files


Please ensure that you open the terminal within the squash-compressor directory before executing the commands.
